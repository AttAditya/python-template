# Python Repo Template

AttAditya's Python Repo Template is a boilerplate for Python projects, providing a structured and organized starting point for developers. It includes a predefined directory structure, essential files, and scripts to streamline the development process.

## Installation

```sh
# Clone the repository
git clone https://github.com/AttAditya/<repo>.git
cd <repo>

# Source the development environment
source dev.sh

# Setup and run
dev setup
dev run
```

## About

This project is designed to facilitate the development of Python applications with a focus on modularity and scalability. It provides a clear separation of concerns, allowing developers to easily manage different components of their application.
